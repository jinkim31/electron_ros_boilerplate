# Electron-ROS Boilerplate

Boilerplate for electron apps that interact with ROS

## Getting Started

- Start app itself (client)
```shell
# install packages
$ npm install
# start in dev mode
$ npm start
```

- Start ROS web server
```shell
$ roscore
$ roslaunch rosbridge_server rosbridge_websocket.launch
```
